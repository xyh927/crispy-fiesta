# 🌡️ 智能温湿度监控系统 (8051单片机)

## 📋 项目概述
本项目基于**STC89C52RC/AT89S52 8051**单片机，实现了一套完整的智能温湿度监控与控制系统。系统集成了实时时钟、温湿度采集、LCD显示、继电器自动控制、串口通信等功能，适用于温室大棚、实验室、仓储等需要环境监控的场景。

## 🎯 核心功能
| 功能模块 | 说明 |
|---------|------|
| ⏰ **实时时钟** | DS1302芯片，支持年月日时分秒+星期显示，可手动校准 |
| 🌡️ **温湿度采集** | DHT11传感器，每200ms采集一次 |
| 📺 **LCD显示** | LCD1602液晶，显示时间、温湿度、设置界面 |
| 🔘 **按键交互** | 4个独立按键，支持短按/长按/释放检测 |
| ⚡ **继电器控制** | 4路继电器：制冷、加热、加湿、UV灯 |
| 📡 **串口通信** | UART 9600bps，可远程控制UV灯开关 |
| 💾 **数据存储** | AT24C02 EEPROM (I2C接口，预留) |

## 🎮 操作模式说明
### 模式0 - 正常显示模式 (默认)
```
┌─────────────────┐
│ 24:10:30 Fri 80%│  ← 第1行: 日期 + 星期 + 湿度
│ 09:13:53  25°C  │  ← 第2行: 时间 + 温度
└─────────────────┘
```
- 自动显示实时时间和温湿度
- 按 **K1** 进入设置模式
### 模式1 - 时间设置模式
```
┌────────────────┐
│ 24-10-30  Fri  │  ← 选中项以1Hz闪烁
│ 09:13:53       │  ← K3减/K4加, K2切换选择项
└────────────────┘
```
- **K2**: 切换设置项 (年→月→日→星期→时→分→秒)
- **K3**: 减1 (秒位清零)
- **K4**: 加1 (秒位清零)
- 松开K3/K4自动保存到DS1302
### 模式2 - 温湿度阈值设置模式
```
┌────────────────┐
│ T_H:30  T_L:20 │  ← 温度上限/下限
│ H_H:60  H_L:40 │  ← 湿度上限/下限
└────────────────┘
```
- **K2**: 切换选择项 (T_H→T_L→H_H→H_L)
- **K3**: 减1
- **K4**: 加1
## ⚡ 继电器自动控制逻辑
### 制冷控制 (RELAY_COOL)
温度 > 上限+1  或  湿度 > 上限+2  →  强制开启制冷
温度 < 上限-1                    →  关闭制冷
### 加热控制 (RELAY_HEAT)
温度 < 下限-1  →  强制开启加热
温度 > 下限+1  →  关闭加热
### 加湿控制 (RELAY_HUMI)
湿度 < 下限-2  →  强制开启加湿
湿度 > 下限+2  →  关闭加湿
### UV灯控制 (串口命令)
串口发送 '#'  →  UV灯开启
串口发送 '$'  →  UV灯关闭


## ⏱️ 定时器中断调度
Timer0 配置为 **10ms中断一次** (11.0592MHz晶振)

## 💡 技术亮点

1. **按键消抖与长按检测**  
   采用状态机方式检测按键，支持短按、长按、释放三种事件
2. **带滞回的继电器控制**  
   设置±1~±2的回差区间，避免继电器频繁切换
3. **非阻塞式传感器读取**  
   DHT11读取时关闭全局中断防止时序被打断，读取完成后恢复
4. **模块化分层设计**  
   驱动层→功能层→应用层清晰分离，便于维护和移植
5. **串口远程控制**  
   支持通过串口命令远程控制UV灯，可扩展为蓝牙/WiFi控制


## 🔌 串口通信协议
- **波特率**: 9600 bps
- **数据格式**: 8位数据位，1位停止位，无校验
- **输出格式**: `TEMP:25,HUMI:60\\r\\n` (每200ms自动发送)
- **控制命令**:
  - `#` → 开启UV灯，返回 `UV ON`
  - `$` → 关闭UV灯，返回 `UV OFF`
